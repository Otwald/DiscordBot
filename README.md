A small little Discord Bot to build a Bridghe between my Jenco Project and the Community Discord for this. Is mostly made for my own personal use.

## Technologies

Project is created with:
*python v3.8
*discord.py
*python-dotenv
*requests

## Setup

```
pipenv update
pipenv shell
python3 main.py
```
